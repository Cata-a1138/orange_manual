# 入服教程

欢迎您加入我们的服务器，在加入前请确保您满足以下条件

  • 年满16周岁
  • 拥有Steam正版GTA5
  • 在游戏中可以使用麦克风正常交流

# 第一步：下载软件

  • KOOK（原开黑啦）官网：https://www.kookapp.cnopen in new window

    服务器ID：24004165 服务器快捷连接：https://kook.top/0tzVJBopen in new window

  • FiveM 官网：https://fivem.netopen in new window 中文网：https://fivem.net.cnopen in new window

# 第二步：面试

  • 面试时间：每天下午3:00-7:00
  • 建议提前在QQ群与管理员取得联系，预约面试时间。
  • 面试前请确保麦克风清晰可用，全程开启，以免耽误时间。

# 第三步：通过白名单

  1. 打开FiveM（无需登录）
  2. 按F8打开控制台
  3. 输入connect cfx.re/join/ld8rmj后按回车
  4. 按F8关闭控制台
  5. 将你的白名单ID私信发给服主
