# 快捷轮盘

使用方法：在步行状态按住F1拖动鼠标到按钮上并点击

  • 更衣：打开更衣菜单

  • 刷车菜单：和[键效果相同

  • 上铐：铐住离你最近的玩家

  • 拉拽嫌犯：可将嫌犯移动到你想要的地方（按F10松开，靠近车辆按E让嫌犯上/下车）

  • 上下班：只有在上班状态才能发送/接收到Code99信息

  • Code99：发送该信息后，所有上班的玩家都将自动导航到发送信息的位置

    如果您距离Code99地点较远，建议先在地图上进行标记，避免Code99信息消失后找不到位置

  • 执法记录仪：打开后您的执法过程将被全程记录！

# 背包

使用方法：在步行状态按F2打开菜单

显示你拥有的物品、金钱

靠近其他玩家时进入现金选项可进行交易

# 玩家列表

使用方法：按F3打开/关闭菜单

显示当前在线的玩家及其ID、延迟

提示

查看自己的ID有三种方法：

  1. 通过右上角信息直接查看（有时会被遮挡）
  2. 按F3找到自己的注册名，查看对应ID
  3. 在聊天框输入命令/info

# 交通管制

使用方法：在步行状态按F5打开菜单

  • 放置物品

  • 交通管制

    添加管制：Range（生效半径）Speed（行驶速度）Create（创建）

    Existing Zones：删除管制

# 职业菜单

使用方法：在步行状态按F6打开菜单

NYPD

  • Citizen Interaction（公民互动）

    ID Card（查看离你最近的玩家信息）

    Search（搜查离你最近的玩家）

    Fine（给离你最近的玩家开罚单）

  • Vehicle Interaction（车辆互动）

    Vehicle Info（查看离你最近的车辆信息）

    Lockpick Vehicle（解锁离你最近的车辆）

    Impound vehicle（扣押离你最近的车辆）

    Vehicle information（查询车辆信息）

  • Object Spawner（放置物品）

# vMenu

使用方法：在步行状态按Insert打开菜单

严禁反RP违规使用！

玩家相关选项

  • 玩家选项

    治愈玩家（恢复玩家生命值）

    装甲类型（恢复玩家护甲值）

    清洁玩家衣服（清除玩家血迹等）

    自杀（一般用于卡Bug时）

  • 玩家出场

    踏板自定义（修改玩家服装）

    保存网页（保存玩家当前服装）

    已保存的踩脚（生成已保存过的服装）

  • 人物模型定制

    可重新捏脸

  • 武器选项

    用于生成武器

  • 武器负荷

    保存加载（保存当前拥有的武器）

    管理负载（生成已保存的武器）

载具相关选项

  • 载具选择

    修理车辆

    设置牌照文字（一般为你的警号，如果是在车店购买或赞助私家车请不要修改，否则会导致无法保存该车辆）

  • 私人载具

    设置车辆（设置后靠近载具双击E可上锁/解锁）
