# 无线电使用手册

# 0.引言

无线电是警员和其他单位交流的主要途径，本手册用于新玩家的快速学习与参考。请注意，无线电频道必须要保持一个良好清楚的环境，以在紧急情况进行沟通，任何人不得占线导致消息传达不清晰。

# 1.服务器常用代码

# 1.1服务器常用Code代码

Code2 常规事件（非紧急），不需要使用警灯和警笛

Code2-H 优先事件（紧急），需要使用警灯

Code3 紧急事件，需要使用警灯和警笛

Code4 案件结束无需进一步增援

Code6 离车进行调查

Code6-A 离车调查,需要增援

Code7 用餐休息（常用于游戏时有事离开）

Code11 请求SWAT

Code99 紧急情况（Code99响起，一切巡逻状态的警员都务必响应）

# 1.2服务器常用10代码

10-4 收到

10-8 返回巡逻状态

10-41 上班

10-42 下班

# 无线电汇报格式

# 2.1个人呼号与单位简称

例：5-L-34，1-A-75，6-M-83

5（代表所属部门）-L/A/M/W（代表单位性质）-34（个人编号）

目前服务器内，1代表密申罗警局，5代表洛杉矶郡警察局，6代表好莱坞警局

L （Lincoln）（音译：林肯）：单人巡逻单位

A（Adam）（音译：艾达）：双人巡逻单位

B（Boy）：三人巡逻单位

M（Mary）（音译：麦瑞）：摩托骑警单位

W（William）（音译：威廉）：警探单位（不穿制服开无标识的请用它）

因此：5-L-34=洛杉矶郡警察局单人巡逻单位34号；1-A-75=密申罗警局双人巡逻单位75号；6-M-83=好莱坞警局骑警单位83号

其他单位：RA=消防救援单位；EMS=急救单位

# 2.2无线电检查格式

警员1：Radio check（无线电检查）

警员2：Radio clear（清晰）（只需一人回复）

# 2.3案件响应格式

调度：各单位注意，这里是Dispatch （调度中心），Location （位置）804发生了一起民事纠纷，需要一个单位以Code2-H响应

警员：5-L-34，Roger that（收到），Code2-H响应Location（位置）804的案件

# 2.4到达报告格式

警员：5-L-34，已抵达8043，Show me code6 （到达现场下车）或 code6-A（到达现场下车并需要支援）

# 2.5交通截停汇报格式

警员：Dispatch（调度中心），5-L-34，Location（位置） 804截停一辆黑色四门轿车，车牌号为SBT80BVM，请帮我查询车辆信息。

调度：收到。正在为你查询。车牌号SBT80BVM在（某人）名下，此人无犯罪记录，无持枪证，车辆保险在有效期内，车辆无被盗报告。

警员：Roger that（收到），show me code6 （到达现场下车）或 code6-A（到达现场下车并需要支援）

截停检查完成后：

警员：Dispatch（调度中心），5-L-34，位于8047的交通截停案件Code4。

# 2.6追捕发起格式

警员：Dispatch（调度中心）5-L-34 location（位置）101发起追捕一辆黑色四门轿车，车牌号为SBT80BVM，涉嫌拒捕，逃逸，车上人员一名，武装程度不详，all units（所有巡逻单位） 监督组单位 空中支援单位 更多支援单位 Code3！

# 2.7追捕进行格式

参与追捕：Dispatch（调度中心）5-L-21 加入追捕队列，目前位于队列第X位。

需要超越前方车辆：5-L-21，前方1150号探险者，注意，我将从你左侧超车。

申请执行PIT：Dispatch（调度中心）5-L-34，location（位置） 高速路935处，嫌犯拒绝停车并多次危险驾驶冲撞警车，目前我方车速65，对方车速70，周围环境安全，申请执行PIT。

# 2.8案件报告格式

大致逻辑为：我在呼叫谁+我表明身份+我在哪+我需要什么

举例1：Dispatch（调度中心），5-L-34，location（位置）100发现一名活跃枪手，需要更多单位支援。

举例2：Dispatch（调度中心），5-L-34，location（位置）804，一个男子拦住了我们的警车，Code6 或 Code6-A 。

举例3：Dispatch（调度中心），5-L-34，location（位置）9021发生劫案，需要更多单位增援，以及一名警司到场。

举例4：5-L-34，shots fire！shots fire！警员倒地！location（位置）702，Code99！需要支援！（紧急情况下可以忽略呼叫方）

举例5：Dispatch（调度中心），5-L-34，location（位置）7027 嫌犯被击毙，现场需要法医单位到场，案件Code4。

# 2.9设备使用报告

大致逻辑为：呼号+位置+部署某物

举例：5-L-34，我正在于927与930之间道路部署钉刺带，请各单位注意避让。

# 2.10上班下班格式

上班：1-L-92，10-41

下班：1-L-92，10-42或Code7
